# qb-traphouse
 
 old qb-traphouse was boring and didnt offer much, I decided to create a whole new one with the same sorta look and feel, you can wash your money in the washer and sell illegal items on the laptop. Works with bt-target if you want, just need to enable the code. I have plans to add-on to this script and maybe even add another deeper level to it only with a key. If you find any errors please make a PR <3
